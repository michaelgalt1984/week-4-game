Crystal Collector

Welcome to Crystal Collector! You see four crystals displayed below. Each of those crystals is assigned a different value, which you can't discover until you click on each crystal... Now, pay attention, because while the counter will show you how much that crystal is worth (by increasing by the amount set to that crystal), you will have to remember the value of each individual crystal...


Why must you remember how much each crystal is worth? Because your goal is to match the value displayed as the target value. If you exceed that value... you lose! Each time you match the value, you earn a point in the "wins column"... and each time you exceed the value, you earn a point in the losses column.


The value to match changes every game, as do the values of the crystals, so you can't use the old maths to do the new tricks! Sometimes you just won't be able to win... because if the crystal values can't be added up to equal the target value... you will never hit it square on the head.


Good luck and have fun!
